What is CSS ?

  1. CSS stand for Cascading Style Sheet.
  2. CSS describe how HTML elements are to be display on screen,paper or in other media.
  3. CSS is not neccesary CSS saves a lot of work.it can control the lay out of multiple web pages all at once.
  4. External stylesheets are stored in CSS file.

CSS can be add to HTML Documents in 3 ways:

  • Inline- by using the style attribute inside HTML elements
  • Internal- by useing style element in the head section
  • External- by using a link element to link to an external CSS file.
  • THe most common way to add CSS,is to keep the style in external CSS file. However, in this tutorial we will use inline and internal styles,because this is easier to demonstrate,and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is CSS class applies a style to multiple elements.ID,on the other hand,applies a style to one unique element. ID is also special in that you can ue a special URL ti link directil to an element and it's used by javascript.

In CSS ,selectors are used to target a specific element or range of elements on a web page.Once an element has been targeted, a style or set of styles can be applied to the element.
